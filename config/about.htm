ELLE

Descripción:

ELLE (Easy Load Layer Extension) permite la carga y guardado de mapas personalizados desde una base de datos PostGIS.

Características:

  • Carga de mapas predefinidos
  • Carga y guardado de leyendas (gvl, sdl) almacenadas en la base de datos
  • Carga automatizada de servicios WMS
